# QUOTIA
- This is a webpage that works by converting input data(Name and age) from the user and displays a welcoming message, their age in months, a message telling them if they are
valid for adult content and a motivational message that repeats itself five times

## Author
Albert Junior Quarshie

## Technology Used:
1. HTML - Basic Structure
2. CSS - Styling of the Webpage
3. JavaScript - Works out the functionalty of the elements of the webpage

## Code Explanation
- Used a form with name and age input for collection of data
- The data is stored in the object localStorage which is also used to perform other logic like displaying the name and age calculation
- Used an if statement in the age calculation sice there can't be a negative number so it does not display any content
- The content is rather displayed in the ageinmonth section that shows it is invalid and also displays the
 confirmation if they are eligible for adult content.
- Finally it displays a motivational quote at the end which is repeated 4 more times on its own seperate line.
